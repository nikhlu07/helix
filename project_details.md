# H.E.L.I.X. - Project Details

## Project Description (Max 100 words)
H.E.L.I.X. (Hedera Enhanced Ledger for Integrity & eXchange) is a decentralized procurement platform designed to eradicate corruption in government spending. Built on Hedera Hashgraph, it combines immutable ledger transparency with a dual-AI fraud detection engine to track public funds in real-time. By tokenizing budgets and automating oversight via smart contracts, H.E.L.I.X. empowers citizens to audit spending and ensures every dollar reaches its intended purpose, restoring trust through verifiable accountability.

## Tech Stack
### Core Blockchain
- **Hedera Hashgraph:** Primary network for consensus and state.
- **Hedera Smart Contract Service (HSCS):** For logic and budget management.
- **Hedera Consensus Service (HCS):** For immutable audit logs.
- **Solidity:** Smart contract language.

### Frontend
- **React:** UI library.
- **TypeScript:** Type-safe development.
- **Vite:** Build tool.
- **Tailwind CSS:** Styling framework.
- **Framer Motion:** Animations.
- **Lucide React:** Icons.
- **HashConnect:** Wallet integration.

### Backend & AI
- **Python (FastAPI):** Backend API server.
- **Node.js:** Deployment and utility scripts.
- **OpenAI (GPT-4):** Natural language processing for fraud analysis.
- **Scikit-learn:** Anomaly detection (Isolation Forest).
- **LangChain:** AI orchestration.

### Infrastructure
- **Git:** Version control.
- **HashPack:** User wallet.
